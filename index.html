  • Deploys are fast

Alternatives

  • Cobbler

  • Ironic?

Ironic standalone + Ansible = Bifrost

Excerpt from Bifrost documentation:

Bifrost is a collection of Ansible playbooks to install Ironic in standalone mode and enroll and deploy servers

Bifrost installs Ironic in a stand-alone fashion:

  • No Keystone needed, Ironic API access allowed without password
  • No Neutron needed, dnsmasq provides DHCP/PXE service and static IP injection is done via config-drive
  • Deployments are performed with the IPA (Ironic Python Agent)

How it works?

  1. Configure global vars to setup environment
  2. Create baremetal inventory
  3. Run install playbook
  4. Run enroll playbook
  5. Run deploy playbook
